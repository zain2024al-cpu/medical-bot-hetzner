## 🏗️ معمارية النظام

### Layers

```
┌─────────────────────────────────┐
│  Telegram Bot Handler           │ (2 زرار)
├─────────────────────────────────┤
│  Report Engine (Facade)         │ (محرك موحد)
├────────┬──────────┬─────────────┤
│ Collect│ Aggregate│ Calculate   │ (3 مراحل)
├────────┴──────────┴─────────────┤
│  Filters (Composite)            │ (9 فلاتر)
├─────────────────────────────────┤
│  Database (Repositories)        │ (SQL)
├─────────────────────────────────┤
│  ReportData (Dataclass)         │ (بيانات)
├─────────────────────────────────┤
│  PDF Builder                    │ (محرك PDF)
├────────┬──────────┬─────────────┤
│ Styles │ Tables   │ Charts      │ (3 مكونات)
├─────────┴──────────┴─────────────┤
│  Export Factory (Plugin)        │ (PDF/Excel/CSV)
├─────────────────────────────────┤
│  Output (File/Stream)           │ (النتيجة)
└─────────────────────────────────┘
```

### Patterns المستخدمة

```
✅ Facade Pattern         (ReportEngine)

✅ Composite Pattern      (CompositeFilter)
✅ Template Method        (PDFBuilder)
✅ Builder Pattern        (ReportData)
✅ Decorator Pattern      (ArabicRenderer)
```
